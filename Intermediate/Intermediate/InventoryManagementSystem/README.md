# Inventory Management System (Java GUI)

## Description
This project is an intermediate-level Java application that
implements an Inventory Management System using Java Swing.

## Features
- Add inventory items
- Update inventory items
- Delete inventory items
- Display inventory list in GUI

## Technologies Used
- Java
- Swing (GUI)
- Collections (ArrayList)

## How to Run
1. Compile InventoryManagementSystem.java
2. Run the program
3. Use the buttons to manage inventory items
