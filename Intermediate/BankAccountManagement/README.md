# Bank Account Management System (Java)

## Description
This project is an intermediate-level Java application that implements
a Bank Account Management System with unit testing using JUnit.

## Features
- Deposit money
- Withdraw money
- Check balance
- Exception handling
- Unit testing using JUnit

## Technologies Used
- Java
- JUnit 5
- OOP Concepts

## Project Structure
- src/BankAccount.java
- test/BankAccountTest.java

## Note
JUnit tests are provided for validation.
